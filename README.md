# 🎵 音频定时播放器

**Windows 单文件版，双击即用，零依赖，零配置**

---

## ✨ 功能特性

- ✅ **单文件 exe**，无需安装，双击直接运行
- ✅ **零依赖**，不需要 .NET、Java、Node.js 等任何环境
- ✅ 支持多个独立定时任务
- ✅ 每个任务可指定音频文件、开始时间、播放时长、重复星期
- ✅ 任务启用/禁用开关
- ✅ 音量独立控制
- ✅ 📱 **手机远程控制**，同 WiFi 下浏览器直接访问管理
- ✅ 数据自动保存在程序目录，U盘拷走就能用
- ✅ 自动扫描 audio 目录的音频文件（mp3、wav、m4a、ogg、flac）

---

## 🚀 使用方法

### 第一步：解压运行
1. 下载 `音频定时播放器.zip`
2. 解压到任意文件夹
3. 双击 `音频定时播放器.exe` 运行

### 第二步：添加音频文件
- 把需要播放的音频文件放入解压目录的 `audio/` 文件夹
- 支持格式：mp3、wav、m4a、ogg、flac

### 第三步：添加定时任务
1. 点击「添加定时任务」按钮
2. 填写任务名称、选择音频文件、设置时间
3. 保存即可

---

## 📱 手机远程控制

1. 确保手机和电脑连接 **同一个 WiFi**
2. 程序界面显示手机访问地址，例如：`http://192.168.1.100:8765`
3. 手机浏览器打开该地址即可管理定时任务

## 🔧 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Rust + Tauri
- **音频播放**: rodio
- **定时调度**: tokio-cron-scheduler
- **远程访问**: Warp Web 服务器

## 📦 打包说明（开发用）

在 Windows 环境下执行：

```bash
# 安装依赖
npm install

# 开发运行
npm run tauri dev

# 打包生产版本（单文件 exe）
npm run tauri build
```

## ⚠️ 注意事项

1. **蓝牙音响**：请先将电脑蓝牙连接到音响，音频会自动通过默认音频设备输出
2. **防火墙**：首次运行可能弹出防火墙提示，请允许访问，否则手机无法远程连接
3. **后台运行**：程序最小化后会继续在后台运行，定时任务正常执行
4. **开机自启**：如需开机启动，请将 exe 快捷方式放入「启动」文件夹

## 🆘 常见问题

**Q: 为什么手机打不开访问地址？**
- 检查手机和电脑是否在同一个 WiFi
- 检查电脑防火墙是否阻止了程序
- 确认端口 8765 没有被其他程序占用

**Q: 为什么没有声音？**
- 检查电脑音量是否开启
- 检查默认音频设备是否正确（蓝牙音响是否连接成功）
- 点击音频文件后的「播放」按钮测试

**Q: 定时任务不执行？**
- 检查任务是否「已启用」
- 检查星期选择是否正确
- 检查电脑时间是否准确
